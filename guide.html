🎓 數位人文專案：雲端與託管全紀錄

這份指南將帶領你完成網頁作品的「GitHub 掛載」與「Firebase 雲端連接」。

📅 第一階段：GitHub 網站託管

1. 安裝工具

  • 下載並安裝 GitHub Desktop。
  • 登入你的 GitHub 帳號。

2. 連結資料夾

  1. 打開 GitHub Desktop，點選 File > Add local repository...
  2. 選擇專案資料夾後點擊 Add repository。
  3. 若提示非 Git 倉庫，點擊 Create a repository。

3. 上傳與發布

  1. 在 Summary 填寫 "Initial Commit"，點擊 Commit to main。
  2. 點擊上方 Publish repository。
  3. 取消勾選 Keep this code private。

☁️ 第二階段：Firebase 雲端資料庫

1. 建立專案

  1. 前往 Firebase Console。
  2. 點擊「新增專案」並建立 Firestore 資料庫。
  3. 選擇「測試模式啟動 (Test mode)」。

2. 取得 API 金鑰

  1. 在專案設定中點擊 </> 註冊 Web 應用程式。
  2. 複製 const firebaseConfig = { ... }; 的內容。

3. 設定程式碼

打開 firebase-config.js，將內容貼入對應位置。

💡 為什麼要這樣做？
GitHub 是你的免費伺服器，Firebase 是你的雲端資料庫。兩者結合，你的作品就能在各處被即時存取！
